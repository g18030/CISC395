# Project Instructions

This project contains various coding exercises, data files, and research papers.

## Directory Structure
- `code/`: Python scripts for sorting and calculation.
- `data/`: CSV files containing student grades.
- `messy/`: Miscellaneous files requiring organization.
- `papers/`: Research papers and documentation.

## Standards
- Adhere to PEP 8 for Python code.
- Ensure all data processing scripts include error handling for missing files.
